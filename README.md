# jamming
-An app to create, view and manage your Spotify playlists
-
Built as a portfolio project for the Codecademy Full Stack Developer course, jamming is a React app that integrates with Spotify to let users manage their Spotify playlists externally. Project requirements specified were building a web app using React, version controlling the application with Git and hosting the repository on GitHub, integrating with Spotify, deploying the application and writing a README file.

-Features:
-
-Search for songs by title, album and artist name

-Add songs to playlist

-Remove songs from playlist

-Rename playlist

-Save playlist to user's Spotify account

-Potential improvements:
-
-Include preview samples for each track

-Only display songs not currently present in the playlist in the search results

-Add a loading screen while playlist is saving

-Update the access token logic to expire at exactly the right time, instead of setting expiration from when the user initiates their next search

-After user redirect on login, restoring the search term from before the redirect

-Ensure playlist information doesn’t get cleared if a user has to refresh their access token
